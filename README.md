# 기획의도 및 컨셉
![image](https://github.com/user-attachments/assets/0e0e935d-dd1b-4d68-95a0-a887c17f7757)
> 주제 : 오락형 결제 보상 서비스
> - 합리적 소비와 앱테크 등에 관심이 많으며 소비에 있어서 재미와 같은 가치를 추구하는 MZ세대를 위한 구독 서비스
> - 본인의 소비 목표를 정하고, 목표를 달성할 시, 보상을 제공하는 서비스 제공
> - 오락적 요소를 가미해, 특정 가맹점에서 불특정 다수가 결제하여 소비 목표 금액을 달성하면 함께 보상을 나눠 갖는 서비스 제공
